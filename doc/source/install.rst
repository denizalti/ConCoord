Installing ConCoord
===================

Requirements
------------------------
The minimum requirements for ConCoord are::

  - python 2.7.2 or later
  - dnspython-1.9.4
  - msgpack-python

Installation
------------------------
ConCoord can be installed from source with:

.. sourcecode:: console

  $ python setup.py install

ConCoord is also available for install through PyPI:

.. sourcecode:: console

  $ pip install concoord
